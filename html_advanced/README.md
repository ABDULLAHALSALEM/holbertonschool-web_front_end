# HTML Advanced

This project contains advanced HTML exercises for learning web development fundamentals.

## Files

- `0-index.html` to `39-styleguide.html` - Progressive HTML exercises
- `index.html` - Main homepage
- `about.html` - About page
- `article.html` - Article page
- `contact.html` - Contact page
- `latest_news.html` - Latest news page
- `styleguide.html` - Style guide
