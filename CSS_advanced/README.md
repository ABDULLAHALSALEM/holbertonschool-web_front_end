## Project Overview

This project focuses on advanced CSS techniques and styling for web development. It builds upon the HTML structure created in the `html_advanced` directory and adds comprehensive styling using modern CSS practices.

## Description

The CSS Advanced project is part of the Holberton School web front-end curriculum. It demonstrates proficiency in:

- Advanced CSS selectors and specificity
- CSS Variables (Custom Properties)
- Flexbox and Grid layouts
- Responsive design principles
- Typography and font styling
- Color theory and design systems
- Box model and positioning
- CSS animations and transitions
- Pseudo-classes and pseudo-elements
- Modern CSS best practices

## Features

- **Progressive Enhancement**: Each CSS file builds upon the previous one
- **Responsive Design**: Mobile-first approach with media queries
- **CSS Variables**: Using custom properties for maintainable code
- **Modern Layout**: Flexbox and CSS Grid implementations
- **Typography**: Custom font stacks and text styling
- **Visual Effects**: Hover states, transitions, and animations
- **Style Guide**: Comprehensive styleguide.html demonstrating design system

## Technologies Used

- HTML5
- CSS3
- CSS Custom Properties (Variables)
- Flexbox
- CSS Grid
- Media Queries
- Modern CSS Selectors

## Learning Objectives

After completing this project, you should be able to:

- Understand and apply CSS specificity rules
- Use CSS variables effectively
- Create responsive layouts using Flexbox and Grid
- Implement a consistent design system
- Apply advanced CSS selectors
- Create smooth animations and transitions
- Follow CSS best practices and naming conventions
- Organize CSS code efficiently

## Files Description

### HTML Files
- `index.html` - Homepage
- `about.html` - About page
- `article.html` - Article page template
- `contact.html` - Contact page
- `latest_news.html` - Latest news page
- `styleguide.html` - Design system and style guide
- `*-index.html` - Progressive HTML exercises (0-36)
- `*-styleguide.html` - Progressive styleguide exercises (11, 13, 26, 28, 30, 32-34, 38-39)

### CSS Files
Each CSS file (`1-style.css` through `32-style.css`) represents a progressive step in styling the website, from basic styles to advanced implementations.

## How to Use

1. Clone the repository
2. Open any HTML file in a web browser
3. Each numbered file demonstrates different CSS concepts
4. Review the corresponding CSS file to understand the implementation
